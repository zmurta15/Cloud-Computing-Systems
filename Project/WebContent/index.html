Root for SCC backend

This is a dummy page for SCC backend.

Examples of endpoints that can be called:

  • /rest/ctrl/version
  • /rest/media/{id}
  • /rest/user/ : POST, DELETE, PUT users
  • /rest/user/{id}/listAuctions : GET the list of auctions of the given user
  • /rest/user/auth : POST user's session
  • /rest/auction/ : POST, PUT auctions
  • /rest/auction/{auctionId}/bid : POST bid on the given auction, GET list of bids of the given auction
  • /rest/auction/{auctionId}/question : POST question on the given auction, GET list of questions of given auction
  • /rest/auction/{auctionId}/question/{questionId}/reply : POST a reply on the given question of the given auction
  • /rest/auction/aboutToClose : GET list of auctions that are about to close in less than 30 min
  • /rest/auction/cognitive/{searchQuery} : GET the top 5 searches on the description of the auctions for the string searchQuery using azure cognitive search
  • /rest/auction/cognitive/related/{auctionId} : GET the top 5 related auctions to the given auction using azure cognitive search
